 Task Manager 

 Веб-приложение на Django для управления задачами.

 🚀 Возможности
- ➕ Добавление задач
- ✏ Редактирование
- ❌ Удаление
- 🔄 Изменение статуса (Новая / В работе / Завершена)
- 🔍 Поиск задач
- 📊 Фильтрация по статусу
- 📈 Статистика (общее количество, в работе, завершённые)

---
 🛠 Технологии
- Python
- Django
- SQLite
- HTML / CSS
